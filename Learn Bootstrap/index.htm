По умолчанию в бутстрапе сетка на 12 колонок:

extra small - < 576px - .col-количество колонок. Кроме того, .col- для автоматической разметки колонок на любом разрешении

small - >= 576px - .col-sm-количество колонок

medium - >= 768px - .col-md-количество колонок

large - >= 992px - .col-lg-количество колонок

extra-large - >= 1200px - .col-xl-количество колонок

Ширина полей - 30px (по 15px на каждой стороне)
